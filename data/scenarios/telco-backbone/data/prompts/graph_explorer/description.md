## Foundry Agent Description

> Queries the network topology graph to answer questions about datacenters, routers, firewalls, links, switches, base stations, MPLS paths, services, SLA policies, and BGP sessions. Use this agent to discover infrastructure relationships, trace connectivity paths, determine blast radius of failures, assess SLA exposure, and evaluate datacenter facility dependencies. Does not have access to real-time telemetry, operational runbooks, or historical incident records.
